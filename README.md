# fis-optimizer-php-compactor

fis-plus smarty plugin php compactor 工具

## 如何使用？

```bash
npm install -g fis-optimizer-php-compactor
```

配置 fis-conf.js

```javascript
fis.config.merge({
    modules : {
        optimizer : {
            php : 'php-compactor'
        }
    }
});
```
